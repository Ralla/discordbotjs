# Discordbot for RaA

## Install & setup
* Run `npm i` to install node dependencies (currently tested in version 8 & 9)
* Copy file `.env.example` to `.env` and fill-in bot token and user IDs of admins (if needed)

## Usage
* Now you can run these commands:
 * `npm start`
 * `npm stop`
 * `npm restart`
